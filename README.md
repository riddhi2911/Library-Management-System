# Library-Management-System
Developed a Library Management System to manage books and user activities efficiently. Features include book search, issue/return, due date tracking, and automatic fine calculation. Admin can manage records, while users can view history. Accounts are auto-blocked after three unpaid fines.

## 🚀 Features
- 📖 Book search and availability check  
- ➕ Add, update, and delete book records (Admin)  
- 🔄 Issue and return books  
- ⏰ Due date tracking  
- 💰 Automatic fine calculation for late returns  
- 👤 User registration and login system  
- 📝 User profile management (view & update)  
- 📚 View issued books and history  
- ❌ Remove books from user list  
- 🚫 Auto-block user after 3 unpaid fines


- ## 🛠️ Technologies Used
- 💻 PHP  
- 🗄️ MySQL  
- 🌐 HTML  
- 🎨 CSS  
- ⚡ Bootstrap  
- 🔧 XAMPP Server  


## ▶️ How to Run the Project
1. Install XAMPP/WAMP server  
2. Copy project folder to `htdocs`  
3. Start Apache and MySQL  
4. Import database in phpMyAdmin  
5. Open browser and run: Homepage.php

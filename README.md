# Library-Management-System

## Description
Developed a Library Management System to efficiently manage book records and user activities. The system allows users to search and request books, while the admin can add, update, and delete book details. It includes features such as issuing and returning books, tracking due dates, and automatically calculating fines for late returns. The application also manages user records, monitors book availability, and maintains transaction history, ensuring smooth and organized library operations.

---

## 🚀 Features
- 📖 Book search and availability check  
- ➕ Add, update, and delete book records (Librarian)  
- 🔄 Issue and return books  
- ⏰ Due date tracking  
- 💰 Automatic fine calculation for late returns  
- 👤 User registration and login system  
- 📝 User profile management (view & update)  
- 📚 View issued books and history  
- ❌ Remove books from user list  
- 🚫 Auto-block user after 3 unpaid fines  

---

## 🛠️ Technologies Used
- 💻 PHP  
- 🗄️ MySQL  
- 🌐 HTML  
- 🎨 CSS  
- ⚡ Bootstrap  
- 🔧 XAMPP Server  

---

## ▶️ How to Run the Project
1. Install XAMPP/WAMP server  
2. Copy project folder to `htdocs`  
3. Start Apache and MySQL  
4. Import database.txt in phpMyAdmin  
5. Open browser and run HomePage.php
